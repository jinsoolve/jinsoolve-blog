$101101$ 이런 이진수가 주어질 때 C++에서 1의 개수를 세려면 어떻게 해야 할까?

```cpp
// unsigned int
__builtin_popcount();

// long long
 __builtin_popcountll();
```

그러나 c++20 이후부터는 아래 함수로 가능하는 듯 하다.
```cpp
popcount();
```
